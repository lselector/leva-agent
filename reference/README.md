# Reference – Document Library

This folder is the Layer 3 reference library.
It holds full documents, long-form histories,
detailed specs, and any large reference material.

## Purpose

- Store complete documents too large for memory/
- Keep detailed project histories
- Archive research, specs, and design docs

## Guidelines

- No size limit per file (unlike memory/)
- Use descriptive file names
- Organize into subfolders as needed
- Memory/ files can point here as breadcrumbs

## Examples

- `architecture-history.md`
- `api-design-spec.md`
- `meeting-notes-2026-Q1.md`
