## Task 13 — Chrome Launch Helper Script

### What to do:
Create `chrome_debug` script that starts
Chrome with remote debugging enabled.

### Files to create:
- `chrome_debug`

### Script content:
```bash
#!/usr/bin/env bash
# Launch Chrome with CDP debugging enabled

PORT=9222

# Check if Chrome is already listening
if curl -s http://127.0.0.1:$PORT/json \
   > /dev/null 2>&1; then
    echo "Chrome CDP already running on :$PORT"
    exit 0
fi

echo "Starting Chrome with CDP on :$PORT"
/Applications/Google\ Chrome.app/\
Contents/MacOS/Google\ Chrome \
  --remote-debugging-port=$PORT &

sleep 2
echo "Chrome CDP ready on :$PORT"
```
