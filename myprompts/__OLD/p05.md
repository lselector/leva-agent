## Step 11 — Update Start Script

### What to do:
Update the `start` script to launch both
servers. Keep the old CLI mode available
via `./start cli`.

### Files to modify:
- `start`

### New `start` script:
```bash
#!/usr/bin/env bash
if [ "$1" = "cli" ]; then
  python -m src
  exit 0
fi

uvicorn src.server_llm.app:app \
  --host 127.0.0.1 --port 8000 &
PID_LLM=$!

uvicorn src.server_auto.app:app \
  --host 127.0.0.1 --port 8001 &
PID_AUTO=$!

echo "Jarvis is running:"
echo "  Browser: http://localhost:8000"
echo "  Auto API: http://localhost:8001"
echo "  Press Ctrl+C to stop"

trap "kill $PID_LLM $PID_AUTO 2>/dev/null" EXIT
wait
```

```bash

kill %1

# 2. Test CLI mode still works
```
