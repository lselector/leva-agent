# Memory – Topic Files

This folder holds topic-specific memory files.
Each file is a short (under 4KB) summary of a
specific subject the agent needs to recall.

## Naming Convention

Use descriptive kebab-case names:
- `python-setup.md`
- `project-architecture.md`
- `current-goals.md`

## Guidelines

- Keep each file under 4KB
- Write as concise summaries / breadcrumbs
- Link to reference/ docs for full details
- Update rather than append endlessly
